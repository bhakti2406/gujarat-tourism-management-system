# Gujarat Tourism Management System

A DBMS project that models a tourism management system for Gujarat. The database is designed to manage tourist destinations, visitors, tour packages, accommodations, guides, transport routes, events, bookings, and feedback using a normalized relational schema.

## Features

- Normalized database schema (BCNF)
- ER Diagram and Relational Schema
- MySQL DDL and data insertion scripts
- 25+ SQL queries covering joins, aggregation, subqueries, and nested queries
- Support for tourist spots, packages, bookings, guides, accommodations, transport, and events

## Database Modules

- Tourist Management
- Tourist Spots
- Tour Packages
- Package Bookings
- Accommodations
- Guides & Guide Assignments
- Transport Routes
- Events
- Visits & Entry Tickets
- Feedback System
